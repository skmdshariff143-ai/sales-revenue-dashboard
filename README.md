# 📊 Sales & Revenue Analytics Dashboard

An interactive, self-contained Sales & Revenue Analytics Dashboard built with **HTML**, **CSS**, and **Chart.js** — no build step needed.

## 🚀 Live Demo

**[View Dashboard →](https://skmdshariff143-ai.github.io/sales-revenue-dashboard/)**

## ✨ Features

- **6 KPI Cards** — Total Revenue, Orders, Avg. Order Value, Conversion Rate, New Customers, Net Profit with sparkline trends
- **5 Interactive Charts** — Revenue/Orders trend, Sales by Category (doughnut), Revenue by Region (horizontal bar), Top Products (bar), Profit Margins (radar)
- **Advanced Filtering** — Filter by time period (7D/30D/90D/12M/YTD), region, and product category
- **Sortable Data Table** — 20 recent orders with search, sort, and status badges
- **Premium Dark Theme** — Glassmorphism design with smooth animations and micro-interactions
- **Fully Responsive** — Works on desktop, tablet, and mobile devices
- **Zero Dependencies** — No build step, just open `sales_dashboard.html` in a browser

## 🛠️ Tech Stack

| Technology | Purpose |
|-----------|---------|

| CSS3 | Styling, Animations, Glassmorphism |

## 🏃 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/skmdshariff143-ai/sales-revenue-dashboard.git
   cd sales-revenue-dashboard
   ```

2. **Open in browser**
   ```bash
   # Simply open the file — no server needed!
   start sales_dashboard.html    # Windows
   open sales_dashboard.html     # macOS
   xdg-open sales_dashboard.html # Linux
   ```

3. **Or use a local server**
   ```bash
   npx serve .
   ```
